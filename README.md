* Linguagem DAX
* Excel/SQL

## 🧠 O que foi analisado
*Análise de Performance Global: Desenvolvimento de indicadores para monitoramento do faturamento total e volume de vendas.

*Segmentação de Mix de Produtos: Estruturação de visão por categoria para identificação dos produtos com maior giro.

*Inteligência Logística: Cruzamento de dados geográficos com prioridade de entrega para otimização da cadeia de suprimentos.

*Gestão de Rentabilidade: Monitoramento da média de descontos por subcategoria para controle de margem de lucro.

*Geomarketing: Mapeamento visual das regiões com maior ticket médio para direcionamento de expansão comercial.
